Detailed Description

Boots the Telecom Network Traffic Simulator console experience.

The Main class serves as the minimal entry point that constructs the io.ConsoleUI and hands off execution to its interactive loop. No simulation logic lives here; instead this class simply wires user interaction with the rest of the system.

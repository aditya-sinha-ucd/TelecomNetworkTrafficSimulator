Detailed Description

Encapsulates construction strategies for multiple traffic sources.

Maintains an internally reused list to minimize allocations. Inputs are either explicit Pareto parameters or a SimulationParameters bundle for FGN. Outputs are lists of fully initialized TrafficSource implementations ready for scheduling.

Detailed Description

Console workflow that configures FGN-driven simulations.

Guides the user through configuration, builds model.FGNGenerationParameters, converts them into model.SimulationParameters, and invokes the core.Simulator. Collaborates with ConsolePrompter for input collection and ConfigFileLoader for optional file-based parameters.

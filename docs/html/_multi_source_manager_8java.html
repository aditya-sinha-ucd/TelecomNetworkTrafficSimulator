Detailed Description

Factory-like helper that constructs homogeneous or heterogeneous source pools.

The class centralizes creation of model.TrafficSource instances, supporting both Pareto ON/OFF sources with random parameter variation and model.FGNTrafficSource objects that leverage Fractional Gaussian Noise thresholds. Collaborates tightly with model.SimulationParameters to interpret requested traffic models. Outputs are mutable lists reused by core.Simulator.

Detailed Description

Immutable representation of a discrete simulator event.

Events are the atomic units processed by core.Simulator through the core.EventQueue. Each event indicates when a model.TrafficSource transitions between model.SourceState states, effectively driving the ON/OFF workload generation pipeline. Instances are lightweight value objects ordered by compareTo(Event) for chronological scheduling.

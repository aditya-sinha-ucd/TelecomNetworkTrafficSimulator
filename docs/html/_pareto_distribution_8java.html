Detailed Description

Pareto (Type I) heavy-tailed distribution used for ON/OFF durations.

Provides inverse-transform sampling for generating ON/OFF durations consumed by model.TrafficSource instances. The distribution is parameterized by shape α and scale β, matching textbook Pareto.

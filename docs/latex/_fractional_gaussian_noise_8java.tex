\doxysubsection{Detailed Description}
Fractional Gaussian Noise (f\+Gn) generator using the Davies–\+Harte method. 

Produces Gaussian, long-\/range dependent sequences with configurable Hurst exponent and standard deviation. Emphasizes reproducibility via dependency injection of {\ttfamily \doxylink{}{java.\+util.\+Random}} and guards against numerical instabilities in FFT computations.

\doxysubsection{Detailed Description}
Manages all file output operations for the simulator. 

This class is responsible for creating a new output folder for each run, writing event logs, saving simulation summaries, and handling FGN mode output. Keeping all file operations here keeps the simulator logic clean and organized.

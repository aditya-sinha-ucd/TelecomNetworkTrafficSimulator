\doxysubsection{Detailed Description}
Loads simulation parameters from a simple text or JSON-\/like configuration file. 

Supported formats (key=value per line or JSON-\/style)\+: 
\begin{DoxyPre}
totalTime=1000
numSources=50
onShape=1.5
onScale=1.0
offShape=1.2
offScale=2.0
\end{DoxyPre}
 or 
\begin{DoxyPre}
\{
  "{}totalTime"{}: 1000,
  "{}numSources"{}: 50,
  "{}onShape"{}: 1.5,
  "{}onScale"{}: 1.0,
  "{}offShape"{}: 1.2,
  "{}offScale"{}: 2.0
\}
\end{DoxyPre}
 Lines starting with \textquotesingle{}\#\textquotesingle{} are treated as comments.

\doxysubsection{Detailed Description}
Parses textual configuration files into parameter maps. 

Consumed by the console UI when users opt to load settings from disk. The loader sanitizes input lines, splits on {\ttfamily =} or {\ttfamily \+:}, and handles both plain text and JSON-\/like structures.

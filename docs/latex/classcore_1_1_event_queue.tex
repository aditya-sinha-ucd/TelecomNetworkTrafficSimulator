\doxysubsection{Detailed Description}
Manages the event scheduling and ordering for the simulation. 

This class acts as the central mechanism that determines which event occurs next. It uses a {\ttfamily \doxylink{}{Priority\+Queue}} to ensure events are always processed in chronological order.

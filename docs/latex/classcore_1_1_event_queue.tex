\doxysubsection{Detailed Description}
Thin wrapper around {\ttfamily \doxylink{}{Priority\+Queue}} tailored for simulation. 

Responsible for lifecycle management of scheduled events, providing operations to enqueue, poll, peek, and clear events. The class also exposes diagnostic information such as size for telemetry.

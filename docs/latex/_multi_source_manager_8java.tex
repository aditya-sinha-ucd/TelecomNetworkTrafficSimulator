\doxysubsection{Detailed Description}
Factory-\/like helper that constructs homogeneous or heterogeneous source pools. 

The class centralizes creation of {\ttfamily \doxylink{classmodel_1_1_traffic_source}{model.\+Traffic\+Source}} instances, supporting both Pareto ON/\+OFF sources with random parameter variation and {\ttfamily \doxylink{classmodel_1_1_f_g_n_traffic_source}{model.\+FGNTraffic\+Source}} objects that leverage Fractional Gaussian Noise thresholds. Collaborates tightly with {\ttfamily \doxylink{classmodel_1_1_simulation_parameters}{model.\+Simulation\+Parameters}} to interpret requested traffic models.

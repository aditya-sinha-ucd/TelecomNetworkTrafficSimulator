\doxysubsection{Detailed Description}
Collects and computes key simulation statistics over time. 

The simulator records\+:
\begin{DoxyItemize}
\item Aggregate rate (fraction of active ON sources)
\item Time stamps of samples
\item Summary stats (mean, peak, std deviation) 
\end{DoxyItemize}

Provides methods to export time-\/series and summaries for external plotting.

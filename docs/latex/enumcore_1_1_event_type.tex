\doxysubsection{Detailed Description}
Enum representing the two possible types of simulation events. 

Each event corresponds to a transition in a Traffic\+Source\textquotesingle{}s state. 
\begin{DoxyItemize}
\item ON\+: The source starts transmitting (entering active state). 
\item OFF\+: The source stops transmitting (entering idle state). 
\end{DoxyItemize}

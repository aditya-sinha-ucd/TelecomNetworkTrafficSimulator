\doxysubsection{Detailed Description}
Provides helpers for printing warnings and terminating failures. 

Offers two static methods\+: {\ttfamily \doxylink{classio_1_1_error_handler_a625c308710b6c7707909869a49a7a8df}{handle\+Error(\+String, boolean)}} for fatal/non-\/fatal errors and {\ttfamily \doxylink{classio_1_1_error_handler_aee8449654c86101e376cb43af852c236}{warn(\+String)}} for advisory messages. Inputs are user-\/facing strings; outputs are console messages and optional JVM termination.

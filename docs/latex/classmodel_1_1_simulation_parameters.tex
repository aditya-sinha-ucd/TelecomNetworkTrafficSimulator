\doxysubsection{Detailed Description}
Holds all configuration options required by the {\ttfamily \doxylink{classcore_1_1_simulator}{core.\+Simulator}}. 

The structure covers both the classic Pareto ON/\+OFF model and the optional FGN-\/thresholded model, allowing callers to switch behavior with {\ttfamily \doxylink{enummodel_1_1_simulation_parameters_1_1_traffic_model}{Traffic\+Model}}.

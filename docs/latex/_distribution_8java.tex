Abstract base class for probability distributions used by sources. 

Establishes the contract satisfied by concrete distributions such as {\ttfamily \doxylink{classmodel_1_1_pareto_distribution}{model.\+Pareto\+Distribution}}. Collaborators include {\ttfamily \doxylink{classmodel_1_1_traffic_source}{model.\+Traffic\+Source}}, which relies on {\ttfamily \doxylink{}{sample()}} to determine ON/\+OFF durations.

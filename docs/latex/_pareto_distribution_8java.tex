Provides inverse-\/transform sampling for generating ON/\+OFF durations consumed by {\ttfamily \doxylink{classmodel_1_1_traffic_source}{model.\+Traffic\+Source}} instances. The distribution is parameterized by shape α and scale β, matching textbook Pareto.

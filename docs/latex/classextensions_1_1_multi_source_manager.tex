\doxysubsection{Detailed Description}
Utility responsible for constructing collections of {\ttfamily \doxylink{classmodel_1_1_traffic_source}{model.\+Traffic\+Source}} objects. 

Depending on the {\ttfamily \doxylink{classmodel_1_1_simulation_parameters}{model.\+Simulation\+Parameters}}, it can build either classic Pareto ON/\+OFF sources or {\ttfamily \doxylink{classmodel_1_1_f_g_n_traffic_source}{model.\+FGNTraffic\+Source}} instances.

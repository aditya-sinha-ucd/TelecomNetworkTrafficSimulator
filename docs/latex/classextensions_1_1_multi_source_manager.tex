\doxysubsection{Detailed Description}
Encapsulates construction strategies for multiple traffic sources. 

Maintains an internally reused list to minimize allocations. Inputs are either explicit Pareto parameters or a {\ttfamily \doxylink{classmodel_1_1_simulation_parameters}{Simulation\+Parameters}} bundle for FGN. Outputs are lists of fully initialized {\ttfamily \doxylink{classmodel_1_1_traffic_source}{Traffic\+Source}} implementations ready for scheduling.

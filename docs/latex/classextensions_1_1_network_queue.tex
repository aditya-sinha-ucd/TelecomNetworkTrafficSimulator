\doxysubsection{Detailed Description}
Deterministic single-\/server queue used for coarse congestion modeling. 

Accepts bulk arrivals from {\ttfamily \doxylink{classcore_1_1_simulator}{core.\+Simulator}}, advances service progress over time, and aggregates waiting/system time metrics for diagnostic reporting.

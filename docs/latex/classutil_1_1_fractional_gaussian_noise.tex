\doxysubsection{Detailed Description}
Fractional Gaussian Noise (f\+Gn) generator using the Davies–\+Harte method. 

Produces a Gaussian, long-\/range dependent sequence with Hurst exponent H in (0.\+5, 1.\+0). The implementation follows good SWE/\+OOP practices\+:
\begin{DoxyItemize}
\item clear parameter validation
\item dependency injection of Random (reproducible tests)
\item private helpers for each mathematical step
\item guards to avoid numerical Na\+Ns 
\end{DoxyItemize}

\doxysubsection{Detailed Description}
Represents a single ON/\+OFF traffic source in the simulation. 

Each \doxylink{classmodel_1_1_traffic_source}{Traffic\+Source} alternates between active (ON) and idle (OFF) states. The ON and OFF durations are sampled from Pareto distributions to model self-\/similar network traffic.

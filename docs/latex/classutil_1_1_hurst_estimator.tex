\doxysubsection{Detailed Description}
Static utility class implementing Hurst exponent estimation. 

Uses the classical R/S method\+:
\begin{DoxyItemize}
\item Split the series into segments of size N\+\_\+k
\item Compute R/S for each segment
\item Average R/S over segments
\item Regress log(R/S) against log(\+N\+\_\+k) 
\end{DoxyItemize}

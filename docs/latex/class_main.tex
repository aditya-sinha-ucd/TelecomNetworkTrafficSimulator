\doxysubsection{Detailed Description}
Entry point that launches the {\ttfamily \doxylink{classio_1_1_console_u_i}{io.\+Console\+UI}}. 

The class is intentionally lightweight; its sole responsibility is to instantiate the console user interface which in turn orchestrates configuration loading, simulation execution, and output management. It collaborates directly with {\ttfamily \doxylink{classio_1_1_console_u_i}{io.\+Console\+UI}} and indirectly with the simulator stack through that interface. The method signature mirrors the standard Java launcher contract.

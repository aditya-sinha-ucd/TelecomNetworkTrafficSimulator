\doxysubsection{Detailed Description}
\doxylink{class_main}{Main} controller for the event-\/driven telecom traffic simulation. 

Responsibilities include\+: 
\begin{DoxyItemize}
\item Initializing either Pareto or FGN-\/based traffic sources. 
\item Advancing the {\ttfamily \doxylink{classcore_1_1_event_queue}{Event\+Queue}} and processing ON/\+OFF transitions. 
\item Sampling aggregate activity and exporting statistics. 
\item Coordinating optional queue modeling to approximate downstream congestion. 
\end{DoxyItemize}

\doxysubsection{Detailed Description}
High-\/level controller for the event-\/driven simulation engine. 

Acts as the single entry point for running experiments. It initializes traffic models, schedules events, consumes them in chronological order, updates aggregate statistics, and persists results. The simulator consumes {\ttfamily \doxylink{}{Simulation\+Parameters}} and produces CSV/summary outputs plus queue analytics. It collaborates with {\ttfamily \doxylink{classcore_1_1_event_queue}{Event\+Queue}}, {\ttfamily \doxylink{classcore_1_1_simulation_clock}{Simulation\+Clock}}, {\ttfamily \doxylink{classcore_1_1_statistics_collector}{Statistics\+Collector}}, and IO helpers.

Runs the simulation loop and exports data to organized outputs. 

The method drains the {\ttfamily \doxylink{classcore_1_1_event_queue}{Event\+Queue}} while the {\ttfamily \doxylink{classcore_1_1_simulation_clock}{Simulation\+Clock}} is below {\ttfamily \doxylink{}{total\+Simulation\+Time}}. For each event it updates the corresponding source, schedules the next event, captures samples, and writes to the {\ttfamily \doxylink{interfaceio_1_1_output_sink}{Output\+Sink}}. At completion the method finalizes queue statistics and exports CSV plus summaries.

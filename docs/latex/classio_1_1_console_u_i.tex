\doxysubsection{Detailed Description}
Handles all console interaction with the user. 

Focuses on prompting and dispatching simulation modes while delegating heavy lifting to {\ttfamily \doxylink{interfaceio_1_1_simulation_mode_handler}{Simulation\+Mode\+Handler}} implementations. Input\+: user menu selections and parameters via the {\ttfamily \doxylink{classio_1_1_console_prompter}{Console\+Prompter}}. Output\+: triggered simulation runs and user feedback.

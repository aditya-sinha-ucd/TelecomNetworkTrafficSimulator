\doxysubsection{Detailed Description}
Priority-\/queue backed scheduler used by the simulator core. 

Encapsulates a {\ttfamily \doxylink{}{java.\+util.\+Priority\+Queue}} of {\ttfamily \doxylink{classcore_1_1_event}{core.\+Event}} instances ordered by time. The queue is the primary collaborator of {\ttfamily \doxylink{classcore_1_1_simulator}{core.\+Simulator}}, supplying it with the next chronological event to execute.

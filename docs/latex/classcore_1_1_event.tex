\doxysubsection{Detailed Description}
Represents a discrete event in the simulation. 

Events are processed chronologically by the \doxylink{classcore_1_1_simulator}{Simulator} through the \doxylink{classcore_1_1_event_queue}{Event\+Queue}. Each event is generated by a Traffic\+Source and determines when it switches state (ON or OFF).

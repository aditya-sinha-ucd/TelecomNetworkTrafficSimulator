\doxysubsection{Detailed Description}
Captures the time, source, and type of a discrete simulation event. 

Collaborates with {\ttfamily \doxylink{classcore_1_1_event_queue}{core.\+Event\+Queue}} (for ordering) and {\ttfamily \doxylink{classcore_1_1_simulator}{core.\+Simulator}} (for execution). Inputs include the event timestamp, the originating source ID, and the {\ttfamily \doxylink{enumcore_1_1_event_type}{core.\+Event\+Type}} describing the ON/\+OFF transition. The class outputs this structured information via accessors and comparison logic.

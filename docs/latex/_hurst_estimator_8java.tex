\doxysubsection{Detailed Description}
Computes the Hurst exponent H using classical Rescaled Range (R/S) analysis. 

This implementation fixes segmentation, cumulative deviations, averaging across windows, and linear regression. It produces stable and correct H estimates for long-\/memory processes such as FGN.

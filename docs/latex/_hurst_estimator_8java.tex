\doxysubsection{Detailed Description}
Provides a simple Rescaled Range (R/S) estimator for the Hurst exponent. 

Used to characterize the self-\/similarity of recorded aggregate rates within {\ttfamily \doxylink{classcore_1_1_statistics_collector}{core.\+Statistics\+Collector}} and FGN workflows. Outputs a value in \mbox{[}0,1\mbox{]} indicating persistence or anti-\/persistence.

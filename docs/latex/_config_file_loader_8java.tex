\doxysubsection{Detailed Description}
Utility for reading simulation parameters from simple config files. 

Supports {\ttfamily key=value} and JSON-\/like syntaxes, ignoring comments and formatting characters. Converts numeric values into a {\ttfamily \doxylink{}{java.\+util.\+Map}} so higher layers (e.\+g., {\ttfamily \doxylink{classio_1_1_console_u_i}{io.\+Console\+UI}}) can hydrate {\ttfamily \doxylink{classmodel_1_1_simulation_parameters}{model.\+Simulation\+Parameters}}.

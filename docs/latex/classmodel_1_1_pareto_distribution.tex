\doxysubsection{Detailed Description}
Implements the Pareto (Type I) heavy-\/tailed distribution. 

Probability Density Function\+: f(x) = (alpha \texorpdfstring{$\ast$}{*} scale\texorpdfstring{$^\wedge$}{\string^}alpha) / x\texorpdfstring{$^\wedge$}{\string^}(alpha + 1), for x \texorpdfstring{$>$}{>}= scale 

Used for modeling ON and OFF durations of traffic sources.
